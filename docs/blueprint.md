# **App Name**: Ledna Commodities Platform

## Core Features:

- Landing Page: Landing page with clear navigation to sign-up/login.
- User Accounts: User account creation and management.
- User Dashboard: Dashboard to manage commodity listings and user profile.
- Commodity Upload: Commodity listing upload form with fields for product details and pricing, with seller contact and page link.
- User Profile: User profile section with contact information and location details and picture, it should contain verification feature using NIN or international passport.
- Listing Display: Display a list of commodities uploaded by the user.
- market trends: Show current price of commodities in a chart form
- Reputation and past sales: Page of each seller should contain rating and reputation gotten from people that experienced doing business with him.

## Style Guidelines:

- Primary color: Earthy Green (#8FBC8F) for representing agriculture and nature.
- Background color: Light Beige (#F5F5DC) to create a clean, neutral backdrop.
- Accent color: Golden Yellow (#FFD700) for highlighting important elements.
- Body and headline font: 'PT Sans', a humanist sans-serif suitable for headlines and body text.
- Use relevant icons to depict product types.
- Clean, grid-based layout for easy navigation.
- Subtle transitions for user interactions.
